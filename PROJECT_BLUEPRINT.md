# Rohit Ohal Photography Platform V2

## Vision

Build a premium photography platform that combines a luxury portfolio website,
a powerful admin panel, and a scalable backend.

The goal is that every future update to the website can be done through the
admin panel without editing code.

# Development Rules

- Production quality code only.
- No temporary components.
- Every component must be reusable.
- Every feature must work with the future database.
- Images stored in Cloudinary.
- Database stores metadata only.
- Admin Panel controls website content.
- Never hardcode business content into React components.

---

# Project Goal

The final platform should allow Rohit to:

- Upload new weddings
- Upload gallery images
- Manage homepage
- Publish journal articles
- Manage enquiries
- Update website without editing code
